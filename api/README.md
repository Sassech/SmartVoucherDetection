# API — SmartVoucherDetection

Backend FastAPI para el reconocimiento automático de comprobantes de depósito.

Stack: FastAPI · Pydantic v2 · SQLAlchemy 2 (async) · PostgreSQL · Redis · Celery · llama-server (glm-ocr).

> Para el plan de desarrollo y estado actual, ver `PROGRESO.md` en la raíz del repo.
